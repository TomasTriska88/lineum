# Lineum Merch: Universum Edition

This directory contains the artwork and mockups for the "Universum" merchandise edition.

## Generation Prompts

To reproduce or update these assets in the future, use the following prompts in the AI image generator.

### 1. Base Art (`art.png`)
**Prompt:**
> Raw printable graphic asset. A breathtaking, high-tech abstract visualization of a complex interconnected network of nodes and flowing data streams, representing the 'Lineum Universum' simulation engine. The style should be futuristic, cinematic, and highly detailed, utilizing a color palette of deep space blacks, vibrant cyber-cyan, and electric fuchsia/purple. The network should appear to be glowing and pulsing with energy. PURELY TRANSPARENT background (Alpha channel) for flexible apparel printing.

### 2. T-Shirt Mockup (`mockup_shirt.png`)
**Input Image:** `art.png`
**Prompt:**
> Photorealistic mockup of a premium, slim-fit black cotton t-shirt. On the center chest, precisely print the provided 'Universum' network artwork. The t-shirt is draped elegantly in a sleek, dimly lit studio setting to emphasize the 'quiet luxury' and 'tech-wear' aesthetic. Professional e-commerce product photography. The symbol on the shirt MUST be exactly the provided network image.
